# 💎LunaLux - Boutique de Bijoux en Ligne

## Description

**LunaLux** est une plateforme web dédiée à la vente de bijoux élégants et modernes.
Ce projet a été développé dans le but de créer une expérience utilisateur fluide et esthétique, inspirée de l’univers du luxe.

---

##  Fonctionnalités

* Affichage des produits (bijoux)
* Ajout de produits
* Ajout au panier
* Suppression du panier
* Interface moderne et élégante
* Gestion dynamique avec PHP

---

## Technologies utilisées

* HTML5
* CSS3
* PHP
*Bootstrap
* MySQL (Base de données)
